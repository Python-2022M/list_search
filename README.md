# Problems
## find01_max

  Given the list of numbers, return the maximum number in the list.

**Example 1:**

```Python
Input: [1, 2, 3, 4, 5]
Output: 5

```
**Example 2:**

```Python
Input: list1 = [12, 2, 5, 2, 7, 9, 1]
Output: 12

```
**Constraints:**

  - 2 <= length(list1) <= 10^5
